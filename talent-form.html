There are three options in the system that control how the participants can edit their applications AFTER they are submitted.
  1. A participant can edit their application at any time during the pipeline (We do not recommend this)
  2. A participant can edit their application at any time before the first action is taken, for example when the judging starts
  3. A participant can edit their application up to 30mins after the application has been submitted (Recommended)
Applications can allow multiple people to work on a single proposal in the system. Please note that before people can be included in the team they have to register as well. An alternative to the built-in teams function is to put in a question that asks for the team members information in the proposal form, this is a less restrictive option.
